# website 
This respoitory is the code for the [Floaty Website](https://floaty.fyi).

## Setup

Copy the example.env file to .env and fill in the values.

# Dart installation

You can install Dart from [here](https://dart.dev/get-dart).

## Dependencies

Run `dart pub get` to install dependencies.

## Run

Run `dart run bin/server.dart` to start the site.
